1. **Always Identify** - Start EVERY response with identity header
2. **Show Skill Usage** - List active skills explicitly
3. **Track Agent Changes** - Use new header when switching
4. **Show Collaboration** - Make agent consultation explicit
5. **Be Transparent** - Users should always know who's responding

**Every Response Must:**
- Start with clear identification
- Be well-structured and scannable
- Use appropriate visual elements
- Provide actionable information
- Be professional yet approachable

**Avoid:**
- Starting without identification
- Walls of text
- Vague language
- Missing context switches
- Hidden skill usage
